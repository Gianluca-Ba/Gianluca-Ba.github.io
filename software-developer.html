Population
  • This project is a simple simulation program to calculate the growth of a lama population over time. The user inputs a starting population size and a target population size. The program then calculates the number of years required for the population to reach or surpass the target size based on a simple model: each year, the population grows by one-third of its current size (new births) and decreases by a quarter of its current size (deaths). The program handles fractional llamas by truncating decimals, ensuring only whole llamas are counted. This simple yet e ffective simulation aids in understanding population dynamics over time.

Mario More
  • This project involves creating a text-based version of the pyramid scene from World 1-1 in Nintendo's Super Mario Brothers using the C programming language. The program, named mario, constructs a right-aligned pyramid of hashes (#) to represent bricks. The height of the pyramid is determined by user input, which must be a positive integer between 1 and 8, inclusive. The program prompts the user for this input and keeps re-prompting until a valid integer is provided. Depending on the input, the pyramid's height adjusts accordingly, with each level adding one more hash to create the pyramid shape. This project is an excellent example of using loops and conditional statements in C to handle user input and output a structured text representation.

  • This project involves developing a parser for English sentences using context-free grammar (CFG) in Python, leveraging the nltk library. The main tasks include defining CFG rules in the NONTERMINALS global variable to parse various sentence structures, implementing the preprocess function to tokenize and clean input sentences, and creating the np_chunk function to identify noun phrase chunks within a parsed sentence tree. The objective is to create a parser capable of understanding the structure of sentences, an essential aspect of natural language processing.
